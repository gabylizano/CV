Project idea is tailor resumes using the main.md information and generate a new one.

## General rules

## File Structure

- main.cv file with all the information
- tailored/<job_application>.md

## Rules to write tailored resumes

- Markdown
- Not larger than 2 pages
- Emphasize achievements
- Don't add a key achievements section
- Write achievement to each previous position but don't add a subtitle like "key achievements"
- ATS-Friendly
- My title is "Senior Project & Program Manager"
